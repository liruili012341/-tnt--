# Throw TNT Mod for Minecraft 1.21.1 (Fabric)

这个mod为Minecraft添加了一个新功能：允许玩家使用打火石投掷TNT。

## 功能描述

当玩家背包中有TNT并且手持打火石对着空气长按右键时，会投掷出一个已点燃的TNT。TNT会在2秒后爆炸。此外新增一个生物，掠夺者爆破手，外观与掠夺者一样，可以向村民、玩家、铁傀儡、门投射tnt，不可自然生成。

## 使用方法

1. 确保你的物品栏中有TNT和打火石
2. 手持打火石
3. 对着空气（不要对准方块）长按右键
4. TNT会被投掷出去并在2秒后爆炸
5. 使用/summon throwtnt:pillager_bomber召唤掠夺者爆破手


## 安装说明

1. 下载适用于Minecraft 1.21.1的[Fabric Loader](https://fabricmc.net/use/)
2. 将构建好的mod文件放入 `.minecraft/mods` 文件夹
3. 启动游戏
